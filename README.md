# SDC260 Course Project

This website is the composition of my course project for SDC260, Web Interface Design.

![XAMPP Website Deployment Screenshot](https://ffrenchonionsoupp.github.io/screenshots/deployment.png)

## Description

This website showcases everything we have learned and practiced over the last five weeks. 
Starting with a basic HTML structuring and the linking of a CSS page, each week has brought on
additions and changes to come together in this final product.

## Technologies Used

- HTML5
- CSS3

## Installation Instructions

1. Clone the repository: `git clone https://github.com/ffrenchonionsoupp/ffrenchonionsoupp.github.io
2. Navigate to the project directory: `cd your-repo`
3. Open `index.html` in your web browser.

## Usage

This website requires very little instruction on usage. It is a surface-level portfolio over the basic coding that SDC260 has rewarded us with.
